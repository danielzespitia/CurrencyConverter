# Currency Converter CLI

## Overview

This is a robust command-line interface application developed in Node.js that allows users to perform real-time currency conversions. It integrates with the Frankfurter API to ensure up-to-date exchange rates.

I developed this software to demonstrate proficiency in asynchronous JavaScript programming, recursion, and third-party API integration.

**[Software Demo Video](https://www.youtube.com/watch?v=7can4y8RehM)**

## Development Environment

* **Language:** JavaScript (ES6+)
* **Runtime:** Node.js
* **External Libraries:** * `axios`: For handling HTTP requests to the API.
    * `readline`: For user input handling.

## Program Execution

To run this program, you need Node.js installed.

1. Clone the repository.
2. Install dependencies:

npm install

3. Run the app: 
node index.js
